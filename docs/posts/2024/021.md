### [Nous Chat](https://hermes.nousresearch.com/)

Nous Research 于 2024 年 11 月 12 日推出了两项新项目：Forge 推理 API Beta 和 Nous Chat。

**Nous Chat** 是一个基于 Hermes 3 70B 语言模型的聊天平台，提供简洁的界面，支持线程式对话系统，帮助用户组织思路和项目。用户可以通过系统提示和配置选项，全面控制与 AI 的交互。该平台目前免费开放，网址为 https://hermes.nousresearch.com

> **Forge 推理 API** 旨在增强现有语言模型的推理能力。它集成了 Hermes 模型家族、代理混合（Mixture of Agents）、代码链（Chain of Code）和蒙特卡洛树搜索（Monte Carlo Tree Search）等多项研究成果。该 API 支持多种模型，包括 Hermes 3、Claude Sonnet 3.5、Gemini 和 GPT-4，用户可以选择单一模型或组合多个模型以提高输出多样性。在 AIME 评估中，Forge 增强的 Hermes 70B 模型在推理能力上表现出色，超越了 Google、OpenAI 和 Anthropic 的更大模型。
>
> 目前，Forge 推理 API 正在 Beta 测试阶段，面向一小部分用户开放。测试重点在于验证推理系统的架构，计划根据用户反馈迅速扩展引擎功能。
